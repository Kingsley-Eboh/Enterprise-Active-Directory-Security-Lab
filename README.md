# Active-Directory-Security-Monitoring
A cybersecurity project focused on **monitoring and securing Active Directory (AD) environments**, analysing event logs, auditing Group Policies, and detecting anomalous user activity.

## Project Overview

- Analyzed **Windows Security Event Logs** to monitor login activity, privilege escalations, and anomalous behavior within an on-prem AD environment.
- Examined **Group Policy Objects (GPOs)** to enforce security baselines, audit configurations, and detect potential misconfigurations that could expose the network to insider or lateral threats.
- Provisioned and managed user accounts and systems within AD to simulate enterprise operations, understanding access control, group memberships, and role-based permissions.
- Investigated Windows authentication workflows using **Kerberos** and **NTLM**, detecting failed logins, ticket anomalies, and potential misuse.
- Correlated event log data to map authentication patterns and policy violations, enabling identification of suspicious lateral movement, brute-force attempts, or unauthorized access.
- Documented findings and developed repeatable monitoring workflows to support SOC incident response procedures and knowledge sharing.
